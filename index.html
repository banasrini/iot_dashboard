DashJazz IoT Connector

The Dashboard for the Internet Of Things

PubNub introduces the simplest IoT dashboard that helps you detect online devices(Raspberry Pi, Arduino) and also talk to them bidirectionally, in real-time.

Github Zip Tutorial

DashJazz IoT Connector

This is a working demo of the DashJazz IoT connector. All you need to do is to connect your embedded devices (Arduino/Raspberry Pi), publish messages to the 'iotchannel', and subscribe to 'userz'. You will see the device UUID appear on the right most column under online devices, and also be able to chat with using the chat box below. If you are new to devices such as Arduino and the Raspberry Pi, dont worry! This page elucidates all the details for you to build your own custom dashboard, with links to the GitHub repositories for all the material.

Basic Introduction

The following are the features provided by this dashboard.

  • Send:

    Send() lets you connect to a channel, and to publish messages to every other device listening on that channel

  • Connect:

    Connect() allows you to listen to other devices, be it an Arduino board, a Raspberry Pi or any other embedded device that talks PubNub.

  • Who's online

    What is the point of sending and connecting if you can't see who's online? This simple dashboard lets you check the different devices online at any instant.

    Arduino

    The Arduino is a tool for making computers that can sense and control more of the physical world than your desktop computer. It's an open-source physical computing platform based on a simple microcontroller board, and a development environment for writing software for the board.
    PubNub provides extensive documentation and support for Arduino. You can find the SDK for Arduino here.


    STEP 1: Connect the Arduino to a computer using the USB port. Using an Ethernet shield, connect the Arduino board to the internet. Now download the Arduino IDE for your PC here . Full and comprehensive instructions for installing the software for Windows, Linux and Mac can be found at this site.
